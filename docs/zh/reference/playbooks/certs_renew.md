# 证书续期 (certs_renew.yaml)

`certs_renew.yaml` 用于为 Kubernetes 集群自动续期服务工作证书（**不包含 CA 证书**）。

## 执行流程

1. **全局初始化**
   - 在所有节点上执行 `native/root` 角色。

2. **加载默认变量**
   - 在所有节点上加载 `defaults` 角色。

3. **证书初始化**
   - 在 `localhost` 上执行 `cert/init`，准备证书续期所需的 CA 与配置。

4. **执行续期**
   - 在所有节点上执行 `certs/renew` 角色，自动检测并续期即将过期或已过期的工作证书。

## 说明

- CA 根证书不会被自动续期；如需更换 CA，请手动处理或使用重新创建集群的方式。
- 建议在证书临近过期前提前执行此 playbook，以避免服务中断。
